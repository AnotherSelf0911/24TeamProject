# 24TeamProject
팀 프로젝트

헬스 도우미 프로그램

1. 회원가입
   1-1 아이디 비밀번호 입력 받기
      1-1-1 아이디 중복 확인
      1-1-2 비밀번호 길이 제한 걸고(8~16자) 특정 문자 꼭 입력하게(알파벳, 숫자 등)
   1-2 회원 정보 입력 받기
      1-2-1 회원 나이, 몸무게, 키(숫자만 들어가게), 이름(문자만 들어가게), 성별(버튼) 입력 받기

2. 회원정보
   2-1 회원가입에서 입력 받은 회원 정보 저장
      2-1-1 회원 나이, 몸무게, 키를 기반으로 한 BMI 수치 출력
   2-2 운동 시작일자(회원가입 한 날짜) 출력
      2-2-1 운동 시작한 날짜로부터 몇일 지났는지 or 이 프로그램을 몇일 사용 했는지 출력
   2-3 회원 정보 바꿀 수 있게 가능
      2-3-1 첫날 몸무게, 키 적은건 따로 저장해둠 (나중에 비교하기위해서)
   2-4 회원 등급 추가
      2-4-1 헬린이 -> 킹반인 -> 헬창 -> 언더아머 단속반 순으로 높게
      2-4-2 등급에 따라 운동 관련 물품 구매 할 때 할인률 적용
      2-4-3 등급은 운동 일차에 따라

3. 운동 도우미
   3-1 운동 도우미 처음 들어오면 운동 난이도 상 중 하 선택하게
      3-1-1 회원이 입력한 BMI 수치를 기반으로 운동 난이도 추천하게
      3-1-2 운동 처음 시작하면 해당 운동이 어디에 좋은지 간략하게나마 설명 (2회부터는 설명x)
   3-2 운동은 3가지 구현 (쉬운 맨몸 운동)
      3-2-1 스쿼트
      3-2-2 팔굽혀펴기
      3-2-3 플랭크
         3-2-3-1 플랭크 타이머 정해놓고 알람 설정 (0초 되면 알람 나오게)
      3-2-4 일차가 늘어날수록 갯수 5개~10개씩 늘어나도록 설정
      3-2-5 일차, 난이도에 따라 운동 루틴 설정
   3-3 회원 BMI 수치에 맞게 대충 식단 관리 (칼로리 어느정도~ 이런느낌)
      3-3-1 식단 추천 시스템 디스플레이

4. 운동 관련 물품 판매
   4-1 프로틴, 운동기구 등
      4-1-1 해당 물품 이미지, 가격, 이름 나오게
      4-1-2 물품 클릭시 상세 페이지로 이동
      4-1-3 물품 종류에 따라 프로틴, 운동기구 탭 따로 만들기
      4-1-4 
   4-2 장바구니 시스템
      4-2-1 갯수 지정 가능
      4-2-2 장바구니에 있는 물품들 총 가격 넣기
      4-2-3 해당 물품 이미지, 가격, 이름 나오게
      4-2-4 구매 버튼 누르면 구매 시스템으로 이동
   4-3 찜해놓기 시스템
      4-3-1 따로 찜해놓기 아이콘 누르면 물품 저장
   4-4 구매 시스템
      4-4-1 장바구니에 저장한 물품, 할인률, 총 가격 다 보이게
      4-4-2 주소나 우편번호, 카드 번호대충 입력만 하면 돌아가게
      4-4-3 카드 결제, 모바일 결제 선택 받기
      4-4-4 결제 버튼 누르면 결제 시스템으로 이동
         4-4-4-1 카드 번호 선택시 바로 결제되게
         4-4-4-2 모바일 결제 선택시 QR로 이동 (QR만 나오고 확인 버튼 누르면 결제 완료)
